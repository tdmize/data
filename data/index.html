Stata Commands & Datasets for Courses / Workshops

  • balanceplot - Stata command for plots of standardized imbalance
  • cleanplots - Stata graphics scheme
  • Data
  • desctable - Stata command for table of descriptive statistics
  • irt_coef - Stata command for y* standardized coefficients from IRT models
  • irt_me - Stata command for marginal effects estimates for IRT models
  • lca_entropy - Stata command for entropy fit statistics from LCA models
  • mecompare - Stata package with for comparing marginal effects across models
  • mecompare_exs - Example files for comparing marginal effects across models
  • meinequality - Stata command for marginal effects inequality statistics
  • sgmediation2 - Stata command for Sobel-Goodman tests of statistical mediation in linear regression
  • totalme - Stata command for total marginal effect estimates
  • usetdm - Stata command to download class and workshop datasets
